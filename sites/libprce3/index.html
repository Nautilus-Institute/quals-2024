PRCE - Perl Regular Compatible Expressions

The PRCE library is a set of functions that implement regular expression pattern matching using the same syntax and semantics as Perl 5. PRCE has its own native API, as well as a set of wrapper functions that correspond to the POSIX regular expression API. The PRCE library is free, even for building proprietary software.

PRCE is now used for many high-profile open source projects, including NGINX,

Download

You can download the debian package source from its official home here:

  • pcre3_8.39-16.tar.gz

To build the package you can use the following commands

tar xfv pcre3_8.39-16.tar.gz
cd pcre3-8.39
dpkg-buildpackage -b

Documentation

You can read PRCE2 HTML documentation, or the text version of the PRCE2 man pages. For Perl 5 regular expression syntax, read the Perl regular expressions man page.

Details on PRCE, and, in particular, comparisons to Perl's regular expression semantics, can also be found in the community authored Wikipedia entry on PRCE.
